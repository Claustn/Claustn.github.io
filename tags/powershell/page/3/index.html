A little more fun with the eventlog in PowerShell

2007-06-28
#PoSH  #PowerShell 
Imagine that you want to list, the last 15 events in all the eventlogs on the system, do you have to write several lines of codes to do that, like this below?? get-eventlog -logname Application -newest 15 get-eventlog -logname System -newest 15 get-eventlog -logname Application -newest 15 Of course not, PowerShell is much smarter than that… You can use the ‘foreach’ command. foreach ($eventlog in (get-eventlog -list -asString)) {write-host $eventlog; get-eventlog $eventlog -newest 15}
