DRAM data rentention

2008-02-24
Researchers from Princeton recently published an article about the possibility to extract information from RAM even after the PC has been turned on. In short, modern RAM contrary to popular belief does not instantly loose its contents, it degrades gradually over time. What does this mean ? One security issue that could arise from the fact that most disk encryption software keeps the encryption key and pre computed key schedules in memory for performance reasons, meaning that if an attacker gains access to the machine he can copy these for offline attacks.
